✨ Propriedades CSS

color – Define a cor do texto. color: blue;
margin – Define o espaçamento externo de um elemento. margin: 20px;
font-family – Define a família da fonte do texto. font-family: 'Arial', sans-serif;
font-weight – Define o peso da fonte (normal, negrito, etc). font-weight: bold;
text-align – Define o alinhamento do texto. text-align: center;
display – Define o tipo de exibição de um elemento (por exemplo, bloco, inline, flex). display: block;
width – Define a largura de um elemento. width: 100%;
height – Define a altura de um elemento. height: 200px;
border-radius – Define o arredondamento das bordas de um elemento. border-radius: 10px;
box-shadow – Adiciona uma sombra ao redor de um elemento. box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.3);
text-decoration – Define a decoração do texto (sublinhado, tachado, etc). text-decoration: underline;
opacity – Define a opacidade de um elemento (quanto mais próximo de 0, mais transparente). opacity: 0.5;
background-image – Define uma imagem de fundo para um elemento. background-image: url('imagem.jpg');
position – Define o tipo de posicionamento de um elemento (estático, relativo, absoluto, fixo). position: absolute;
z-index – Define a ordem de empilhamento de elementos (quem fica em cima de quem). z-index: 10;
flex – Define como os itens em um contêiner flexível devem se comportar. flex: 1;
justify-content – Alinha os itens ao longo do eixo principal em um contêiner flexível. justify-content: center;
align-items – Alinha os itens ao longo do eixo transversal em um contêiner flexível. align-items: flex-start;
