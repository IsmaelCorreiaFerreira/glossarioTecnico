📝 Tags HTML

<img> – Define uma imagem. <img src="imagem.jpg" alt="Descrição da imagem">
<a> – Cria um link clicável. <a href="https://exemplo.com">Visite nosso site</a>
<h1> – Define o título principal da página (Cabeçalho de nível 1). <h1>Título Principal</h1>
<p> – Define um parágrafo de texto. <p>Este é um parágrafo</p>
<ul> – Cria uma lista não ordenada. <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> – Cria uma lista ordenada. <ol><li>Primeiro</li><li>Segundo</li></ol>
<div> – Define uma seção genérica de um documento. <div>Conteúdo aqui</div>
<span> – Define uma parte de texto ou conteúdo em linha. <span>Texto em destaque</span>
<h2> – Define um cabeçalho de nível 2. <h2>Subtítulo</h2>
<strong> – Define um texto com ênfase forte (geralmente em negrito). <strong>Texto importante</strong>
<em> – Define um texto com ênfase (geralmente em itálico). <em>Texto enfatizado</em>
<br> – Insere uma quebra de linha no texto. <p>Primeira linha<br>Segunda linha</p>
<iframe> – Cria um quadro embutido para exibir conteúdo externo (como um vídeo). <iframe src="video.html"></iframe>
<form> – Define um formulário para entrada de dados do usuário. <form>...</form>
<input> – Define um campo de entrada em um formulário. <input type="text" placeholder="Digite algo">
<button> – Define um botão clicável. <button>Clique aqui</button>
<table> – Define uma tabela. <table><tr><td>Conteúdo</td></tr></table>
<label> – Define um rótulo para um elemento de formulário. <label for="nome">Nome:</label>
